Printer Options

The printer options allow you to control how your project will be printed. From this dialog you can set the colors for different keyword lists in the same way as in the code editor as well as some common properties. Details about each option are given below.

Syntax - This allows you to set different colors for printing the Keyword lists
Page Numbering - Enables page numbering and set the position, Timestamp will print a timestamp at the bottom of each page.
Page Heading - Enables printing a header at the top of each page, you can choose to print the File Description
Use Colors - This will enable or disable the printing of colors, set this option to off if you are using a B&W printer or you do not wish to use colors.
