File Browser Options 

The File Browser Options are used in conjunction with the Project window when browsing for files. If you choose to find a file using the browser you can take advantage of the folder and file filters available in this option.

You can set up to ten paths in the browse sequence, these paths will be displayed in order when you press the Previous and Next Folder buttons in the File Browser. Set the file types you wish to view in the filter box (note the dot at the end of the array of filters). These filters can be toggled on or off using the Filter button in the browser.
